# Saiket Systems Internship Projects

A collection of 5 Flutter mini-projects built during my internship 
at Saiket Systems (Dec 2025 – Jan 2026).

---

## Projects

### Task 1 — Simple Quiz App
- Displays a question with multiple choice answers
- Submit button checks selected answer and shows Correct/Incorrect result
- **Tech:** Flutter, Dart

### Task 2 — Simple Image Gallery
- Displays images in a GridView layout
- Supports item click and image selection
- **Tech:** Flutter, Dart

### Task 3 — Contact List App
- Retrieves and displays device contacts
- Requests device permissions
- Populates contact names and numbers in a ListView
- **Tech:** Flutter, Dart

### Task 4 — Activity Navigation App
- Navigates between two screens using Flutter routing
- Passes data from Screen One to Screen Two
- **Tech:** Flutter, Dart, Navigation

### Task 5 — Simple Timer App
- Countdown timer with Start and Reset functionality
- Updates UI every second in real time
- **Tech:** Flutter, Dart, Timer
